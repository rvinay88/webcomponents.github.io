Introduction to Polymer: The Next Generation of Web Development

By Eric Bidelman

SFHTML5

April 29th, 2014

Let's stop treating the DOM as one gigantic pile of everything. Web components solve a number of shortcomings in web development. Let's talk about architectural concepts and begin to think about how to componentize the web. To understand Polymer, you need to understand web components.

Eric Bidelman

@ebidel

Eric is a Staff Developer Programs Engineer on the Google Chrome team. His current passion is web components and Polymer. Eric is the author of 'Using the HTML5 Filesystem API' and regularly contributes to html5rocks.com and other open source projects. Prior to Google, Eric worked as a software engineer at the University of Michigan where he designed rich web applications and APIs for the university’s 19 libraries.
